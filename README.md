# Applications Logger (React.js)

### Getting started

Follow the steps below to run the project to create a development build with hot-reload
for quick changes.

##### Build and run locally
```
# Install dependencies
npm install

# Start the project
npm start

# Build the project
npm run build
```

##### Access the project
Visit `localhost:3000`
